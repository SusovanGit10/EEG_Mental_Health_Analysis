## ⚡ Local Setup

### Clone Repository

```bash
git clone https://github.com/SusovanGit10/EEG_Mental_Health_Analysis.git
cd EEG_Mental_Health_Analysis
```

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
python main.py
```

### Frontend Setup

```bash
cd frontend/user
npm install
npm start
```
